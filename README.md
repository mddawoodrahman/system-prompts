## 📌 Overview

A curated open-source collection of **system prompts**, **agent workflows**, and **tool schemas** for modern LLMs including:

* OpenAI (GPT)
* Anthropic (Claude)
* Google (Gemini)
* Cursor, VSCode Agents, and more

This repository serves as a **central knowledge base** for prompt engineering and AI agent design.

---

## 🚀 Why This Repo?

* 📦 Well-structured prompt taxonomy (System, Agent, Tools)
* 🧠 Multi-model coverage (GPT, Claude, Gemini)
* ⚙️ Includes tool schemas + configs
* 🔄 Continuously updated with latest prompts
* 🌍 Open for community contributions

## 🧩 What's Inside

* 🧠 **System Prompts** → Core behavior, policies, constraints
* 🤖 **Agent Prompts** → Planning, execution, workflows
* 💬 **Chat Prompts** → Conversational styles
* ⚙️ **Tool Schemas** → JSON-based tool definitions
* 📦 **Model Configs** → YAML configurations for different models

## 🤝 Contributing

We welcome contributions from the community!

### 📌 Guidelines:

* Follow existing folder structure
* Use proper naming (`Prompt`, `System`, `Agent`, `Tools`)
* Validate JSON/YAML files
* Add versioning if applicable

### 🔧 Steps:

1. Fork the repo
2. Create a branch
3. Add your prompt/tool
4. Submit a Pull Request
